# Guía Práctica para Aplicar y Revisar Reglas de Contrapunto

Esta guía te ayudará a aplicar sistemáticamente las reglas del contrapunto a partir de una progresión armónica dada, verificar tu trabajo y corregir errores comunes.

## 1. Preparación: De la Progresión al Contrapunto

### 1.1 Análisis de la Progresión Armónica

1. **Identifica la tonalidad** de la progresión
2. **Analiza la función** de cada acorde (tónica, subdominante, dominante)
3. **Identifica puntos cadenciales** importantes
4. **Extrae el bajo** como posible cantus firmus o línea de referencia

**Ejemplo:**
Para una progresión en Do mayor: C - Am - F - G - C

- Tonalidad: Do mayor
- Funciones: I - vi - IV - V - I
- Cadencia: Auténtica (V-I) al final
- Bajo: C - A - F - G - C

### 1.2 Selección de la Especie de Contrapunto

Decide qué especie de contrapunto trabajarás:

| Especie | Descripción             | Mejor para          |
| ------- | ----------------------- | ------------------- |
| Primera | Nota contra nota        | Armonización básica |
| Segunda | Dos notas contra una    | Movimiento moderado |
| Tercera | Cuatro notas contra una | Mayor elaboración   |
| Cuarta  | Síncopa (suspensiones)  | Tensión armónica    |
| Quinta  | Florido (combinación)   | Expresión completa  |

## 2. Aplicación Paso a Paso

### 2.1 Primera Especie (Nota contra Nota)

1. **Establece puntos de anclaje:**

   - Inicia con octava o quinta perfecta
   - Termina con octava perfecta (preferible)

2. **Completa la línea melódica:**

   - Usa principalmente terceras y sextas para los acordes intermedios
   - Evita más de tres intervalos iguales consecutivos
   - Mantén un rango máximo de una octava

3. **Verifica el movimiento entre voces:**
   - Prioriza el movimiento contrario
   - Evita quintas y octavas paralelas o directas
   - Después de un salto grande, muévete en dirección contraria

**Ejemplo práctico:**

```
Progresión: C - Am - F - G - C
Bajo:       C - A  - F - G - C
Soprano:    G - E  - A - B - C  (5ª-3ª-3ª-3ª-8ª)
```

### 2.2 Segunda Especie (Dos Notas contra Una)

1. **Establece puntos de anclaje** (como en primera especie)

2. **Añade notas en tiempos débiles:**

   - Tiempos fuertes: Solo consonancias
   - Tiempos débiles: Consonancias o notas de paso

3. **Notas de paso y bordaduras:**
   - Usa notas de paso para conectar consonancias por grado conjunto
   - Introduce bordaduras para crear interés melódico

**Ejemplo práctico:**

```
Progresión: C    - Am   - F    - G    - C
Bajo:       C    - A    - F    - G    - C
Soprano:    G C  - E F  - A G  - B D  - C
            5ª 3ª- 3ª 6ª- 3ª 2ª- 3ª 5ª- 8ª
```

## 3. Lista de Verificación por Especie

### 3.1 Primera Especie

- [ ] **Intervalos armónicos:** Solo consonancias (3ª, 5ª, 6ª, 8ª)
- [ ] **Inicio y final:** Octava o quinta al inicio; octava al final
- [ ] **Movimiento melódico:** Predominantemente por grado conjunto
- [ ] **Quintas y octavas:** No paralelas ni directas
- [ ] **Variedad interválica:** No más de tres intervalos iguales consecutivos
- [ ] **Rango:** No excede una octava
- [ ] **Independencia:** Cada voz tiene contorno propio

### 3.2 Segunda Especie

- [ ] **Tiempos fuertes:** Solo consonancias
- [ ] **Tiempos débiles:** Consonancias o notas de paso
- [ ] **Disonancias:** Solo como notas de paso en tiempo débil
- [ ] **Saltos:** Seguidos por movimiento contrario
- [ ] **Quintas y octavas:** No consecutivas en tiempos fuertes
- [ ] **Ritmo:** Movimiento constante en la voz contrapuntística

### 3.3 Tercera Especie

- [ ] **Primer tiempo:** Consonancia
- [ ] **Tiempos intermedios:** Pueden ser disonantes como notas de paso
- [ ] **Cambio de dirección:** Cada 2-3 notas para evitar monotonía
- [ ] **Disonancias:** Solo como notas de paso o bordaduras
- [ ] **Saltos:** Compensados con movimiento contrario
- [ ] **Ritmo:** Flujo constante de cuatro notas por compás

### 3.4 Cuarta Especie

- [ ] **Preparación:** Consonancia en tiempo débil
- [ ] **Suspensión:** Misma nota ligada creando disonancia
- [ ] **Resolución:** Descendente por grado conjunto
- [ ] **Proporción:** Al menos la mitad son suspensiones
- [ ] **Cadencia:** Suspensión 7-6 o 4-3 antes de la cadencia final

### 3.5 Quinta Especie

- [ ] **Combinación:** Uso efectivo de elementos de todas las especies
- [ ] **Variedad rítmica:** Alternancia de valores rítmicos
- [ ] **Disonancias:** Correctamente preparadas y resueltas
- [ ] **Flujo:** Movimiento continuo con momentos de reposo
- [ ] **Expresividad:** Equilibrio entre reglas y expresión musical

## 4. Detección y Corrección de Errores Comunes

### 4.1 Quintas y Octavas Paralelas

**Error:**

```
Soprano: E - F - G
Bajo:    A - B - C
        5ª - 5ª - 5ª (¡Error!)
```

**Corrección:**

```
Soprano: E - D - G
Bajo:    A - B - C
        5ª - 3ª - 5ª (Correcto)
```

### 4.2 Quintas y Octavas Ocultas

**Error:**

```
Soprano: E - G
Bajo:    C - G
        3ª - 8ª (¡Error! Movimiento directo a octava)
```

**Corrección:**

```
Soprano: E - D
Bajo:    C - G
        3ª - 5ª (Correcto)
```

### 4.3 Disonancias Mal Tratadas

**Error en 2ª especie:**

```
Soprano: E - D - C - B
Bajo:    C - - - G
        3ª - 2ª - 7ª - 3ª (¡Error! Disonancia en tiempo fuerte)
```

**Corrección:**

```
Soprano: E - D - C - B
Bajo:    C - - - G
        3ª - 9ª - 7ª - 3ª (Correcto)
```

### 4.4 Saltos Excesivos o Mal Compensados

**Error:**

```
Soprano: C - G - B
        (Salto ascendente seguido de otro salto)
```

**Corrección:**

```
Soprano: C - G - F
        (Salto seguido de movimiento contrario)
```

## 5. Método de Trabajo con Progresiones Modernas

### 5.1 Adaptación a Progresiones con Acordes Extendidos

1. **Identifica las notas estructurales** de cada acorde (1-3-5-7)
2. **Trata las tensiones** (9-11-13) como posibles disonancias
3. **Aplica las reglas de contrapunto** a las notas estructurales
4. **Integra las tensiones** siguiendo principios de preparación y resolución

### 5.2 Ejemplo con Progresión de Jazz

Para una progresión Cmaj7 - Am7 - Dm7 - G7 - Cmaj7:

1. **Extrae notas estructurales:**

   - Cmaj7: C-E-G-B
   - Am7: A-C-E-G
   - Dm7: D-F-A-C
   - G7: G-B-D-F
   - Cmaj7: C-E-G-B

2. **Crea línea contrapuntística** respetando las reglas
3. **Añade tensiones** como embellecimiento

## 6. Ejercicios Prácticos

### 6.1 Ejercicio Progresivo

1. **Toma una progresión simple** (I-IV-V-I)
2. **Crea un contrapunto en primera especie**
3. **Elabora el mismo contrapunto en segunda especie**
4. **Continúa con tercera y cuarta especies**
5. **Finaliza con un contrapunto florido**

### 6.2 Análisis de Obras Maestras

1. **Selecciona un fragmento** de Bach, Palestrina o Fux
2. **Identifica la especie predominante**
3. **Analiza los intervalos armónicos**
4. **Observa el tratamiento de disonancias**
5. **Nota las excepciones a las reglas**

## Conclusión

El contrapunto es una disciplina que equilibra reglas técnicas con expresión musical. Al seguir esta guía paso a paso, podrás transformar cualquier progresión armónica en un contrapunto bien estructurado y musicalmente satisfactorio. Recuerda que la práctica constante es la clave para dominar este arte.
