# Ejercicios Prácticos: Contrapunto Segunda Especie

Esta hoja de ejercicios te ayudará a practicar y dominar el contrapunto de segunda especie a través de actividades progresivas, desde el análisis hasta la composición completa.

## Ejercicio 1: Identificación de Intervalos y Disonancias

### 1.1 Analiza el siguiente fragmento de contrapunto de segunda especie

```
Contrapunto:  -  G | E  C | F  E | D  C | C
              |    | |  | | |  | | |  | |
Canto dado:   C    | G    | A    | F    | C
```

**Instrucciones:**

1. Identifica todos los intervalos armónicos
2. Marca las disonancias y determina si son correctas o incorrectas
3. Identifica los tipos de movimiento entre cada par de notas consecutivas

### 1.2 Identifica y corrige los errores en el siguiente contrapunto

```
Contrapunto:  -  E | G  F | A  G | F  D | C
              |    | |  | | |  | | |  | |
Canto dado:   C    | C    | D    | G    | C
```

**Errores a encontrar:**

- Disonancias mal empleadas
- Posibles quintas u octavas paralelas
- Saltos no compensados
- Otros errores de conducción de voces

## Ejercicio 4: Composición Guiada

### 4.1 Compón un contrapunto de segunda especie sobre este canto dado en Do mayor

```
Canto dado: C - F - G - E - C
```

**Instrucciones:**

1. Comienza con un silencio seguido de una consonancia perfecta
2. Incluye al menos una disonancia como nota de paso
3. Termina con una cadencia apropiada hacia una octava

### 4.2 Compón un contrapunto de segunda especie sobre este canto dado en La menor

```
Canto dado: A - E - F - G - E - A
```

**Instrucciones:**

1. Coloca el contrapunto por debajo del canto dado
2. Utiliza al menos dos disonancias como notas de paso
3. Incluye un salto compensado por movimiento contrario

## Ejercicio 5: Composición Libre

### 5.1 Crea tu propio canto dado de 5-7 notas y compón un contrapunto de segunda especie sobre él

**Instrucciones:**

1. Elige una tonalidad (mayor o menor)
2. Crea un canto dado melódicamente interesante
3. Compón un contrapunto de segunda especie siguiendo todas las reglas
4. Analiza tu propia composición marcando intervalos y movimientos

### 5.2 Composición a dos voces con cambio de posición

**Instrucciones:**

1. Crea un canto dado de 8 notas
2. Compón un contrapunto de segunda especie para los primeros 4 compases
3. Para los siguientes 4 compases, intercambia las funciones: el contrapunto se convierte en canto dado y viceversa
4. Asegúrate de que la transición entre las dos secciones sea fluida

## Ejercicio 6: Análisis de Repertorio

### 6.1 Busca ejemplos de contrapunto de segunda especie en el repertorio barroco

**Sugerencias:**

- Invenciones a dos voces de J.S. Bach
- Sonatas para violín solo de J.S. Bach
- Dúos de G.P. Telemann

**Instrucciones:**

1. Identifica pasajes que ejemplifiquen contrapunto de segunda especie
2. Analiza cómo los compositores aplican o modifican las reglas
3. Observa cómo se integra la segunda especie con otras texturas contrapuntísticas

## Ejercicio 7: Aplicación Creativa

### 7.1 Contrapunto modal

**Instrucciones:**

1. Crea un canto dado en modo dórico o mixolidio
2. Compón un contrapunto de segunda especie que respete las características modales
3. Reflexiona sobre las diferencias entre el contrapunto tonal y modal

### 7.2 Contrapunto con aplicación contemporánea

**Instrucciones:**

1. Toma una progresión de acordes de una canción popular o de jazz
2. Extrae una línea de bajo o melodía como canto dado
3. Compón un contrapunto de segunda especie que pueda funcionar como contramelodía
4. Adapta las reglas según sea necesario para el estilo musical elegido

## Soluciones Modelo para Ejercicios Seleccionados

### Solución para Ejercicio 1.1

```
Contrapunto:  -  G | E  C | F  E | D  C | C
              |    | |  | | |  | | |  | |
Canto dado:   C    | G    | A    | F    | C
Intervalos:   -  5ª| 6ª 4ª| 6ª 5ª| 6ª 5ª| 8ª
Movimiento:   -    | ↓  ↓ | ↑  ↓ | ↓  ↓ | ↑
```

**Análisis:**

- La 4ª en el compás 2 (tiempo débil) es una disonancia correctamente empleada como nota de paso
- Todos los tiempos fuertes contienen consonancias
- Hay una mezcla adecuada de movimiento contrario y directo
- La cadencia final es correcta con movimiento hacia la octava

### Solución para Ejercicio 2.2

```
Contrapunto:  -  G | E  D | A  G | D  B | C
              |    | |  | | |  | | |  | |
Canto dado:   C    | C    | F    | G    | C
Intervalos:   -  5ª| 3ª 2ª| 3ª 2ª| 5ª 3ª| 8ª
```

**Explicación:**

- D en el compás 2 (tiempo débil): Nota de paso disonante entre E y C
- G en el compás 3 (tiempo débil): Nota de paso disonante entre A y F
- B en el compás 4 (tiempo débil): Consonancia que prepara el movimiento hacia C
