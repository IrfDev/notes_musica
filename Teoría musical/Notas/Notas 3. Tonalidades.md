El armónico es cuando la misma nota tiene nombres diferentes 

Formula escala mayor: 
- TTSTTTS



## Instrucciones para armonización de distintas escalas: 

1. Partir de una de las 12 notas
2. Aplicar la formula para la escala mayor
3. Contar las notas para armar las 5tas y 7mas 
	1. Realizar las modificaciones según la formula para que se ajusten a la formula 
		- Bemol es medio tono para abajo
		- Sostenido es medio tono para arriba

- Tonalidades y armaduras

- Armónico
- Voicing
